Application is written on Node.js 
To launch use cmd command: 
> node main.js

Input names of participants, separated by commas. There should be a power of two.
After you get pairs of participants. You need to input results of their games this way:
Write 'first' if first player won and 'second' otherwise. Write first/second as many as number of pairs, 
separated by commas. 

When all champion stages are gone, you'll get a winner.
