Application is written on Node.js 

Download this directory. Run in cmd next command:
>npm install

To launch the project use next command:
>npm start

Input number of players. It should be a power of two.
Then input names of players.
Choose the way to show tounament bracket (olympics or double bracket).
Choose index of winner (1 or 2) for every game.

When all tournament rounds are played, you get a winner.
